What to Test (2026-07-05)

Build: Home & Lock Screen Widgets Date: July 5, 2026
Build Focus: Introduces Home Screen and Lock Screen widgets (Trip Status + Quick Start).

Open the app once before adding widgets. Note: A full setup guide for widgets is available in the tutorials section.

1. Adding the Widgets

  • Home Screen: Long-press empty spot → Edit/+ → search "SafelySolo" → add TRIP STATUS (small/med) and QUICK START (med/large).
  • Lock Screen: Long-press Lock Screen → Customize → Lock Screen → add circle/rectangle widgets below the clock.

2. Trip Status Widget

  • Start a trip → verify the widget shows destination, arrival time, and a LIVE countdown.
  • Check for "Deadline Soon" (orange) and "Deadline Passed" (red) states.
  • Verify the widget resets to "Start a trip" after checking in safe.

3. Quick Start Widget

  • Tap Trip Home / Round Trip / Place → verify the app opens with the trip details pre-filled.
  • Verify the widget shows a compact countdown when a trip is active.

Known Issue

If a trip completes via arrival detection while the app is in the background, the widget may keep counting down until the app is opened. We're investigating.

Feedback we want

  1. Did the countdown stay accurate with the app closed?
  2. Did the widget reset promptly after you checked in?
  3. Are the Quick Start tiles the places you'd actually want?
